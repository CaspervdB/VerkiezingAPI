Deel 1,2 en 3 Casper van den Berg 4580125

In deze map staan 2 mappen met validatie schema's, 1 voor XML en 1 voor JSON daarnaast kan je de api opstarten door in de backend folder een terminal te openen en hier npm install, npm run build en daarna npm run start uit te voeren en de frontend door npm start uit te voeren in de frontend folder. 
De backend maakt gebruik van een mongoDB database, deze zal in docker draaien.
